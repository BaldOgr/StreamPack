putValue

fun putValue(value: Long, maxValue: Long): BitBuffer

Given the specified maxValue, this method writes the specified value to this BitBuffer using the optimal amount of bits, thus providing free compression.

Parameters

value - the value to write to this BitBuffer.

maxValue - the maximum possible value that value can be; a lower maxValue results in a better compression ratio.

Exceptions

IllegalArgumentException - if maxValue is negative or if the absolute value of value is greater than maxValue.

Return
this BitBuffer to allow for the convenience of method-chaining.
