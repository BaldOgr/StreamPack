getValue

fun getValue(maxValue: Long): Long

Given the specified maxValue, this method reads a value from this BitBuffer using the optimal amount of bits, thus providing free compression. The value of maxValue should be the same as what was used when calling .putValue.

Parameters

maxValue - the maximum possible value that the value being read can be; a lower maxValue results in a better compression ratio.

Exceptions

IllegalArgumentException - if maxValue is negative.

Return
the value read from this BitBuffer as a long.
